# Object recognition project

Put Test_Set Folder in same directory as Final.py
Open Final.py in normal python IDE (I used spyder with Anaconda)

Run Final.py it will systematically itterate thru all the images and after each picture it will give the picture name centroid coordinates and distance as well displaying a plot showing the region selected on the image. The data at the end of each image is displyed using print so will be in the python terminal.

libs required:
	math
	numpy
	matplotlib
	cv2
	skimage
	os
	pandas
